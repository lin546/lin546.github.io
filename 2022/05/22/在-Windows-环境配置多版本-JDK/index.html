在 Windows 环境配置多版本 JDK

Created2022-05-22|Updated2022-05-22|开发者手册
|Post View:

以 Java 8 与 Java 11 为例。

一、下载安装包

下载地址（Zip包）：

  • Java Platform, Standard Edition 8 Reference Implementations

  • Java Platform, Standard Edition 11 Reference Implementations

下载完成后解压到同一目录下：如（E:\developer）。

二、配置环境变量

  • JAVA8_HOME：变量名：JAVA8_HOME，变量值：JDK8的位置(E:\developer\openjdk-8u41-b04-windows-i586-14_jan_2020)

  • JAVA11_HOME：变量名：JAVA11_HOME，变量值：JDK11的位置(E:\developer\openjdk-11+28_windows-x64_bin)

  • JAVA_HOME：变量名：JAVA_HOME，变量值：%JAVAxx_HOME%

  • CLASSPATH ：变量名：CLASSPATH，变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

  • PATH：%JAVA_HOME%\bin与%JAVA_HOME%\jre\bin
